# Accsaber Leaderboard
This is a mod that adds a leaderboard for accsaber (or more accurately accsaber reloaded). 
## Features
This mod adds a leaderboard, as well as a way to track milestones, view other player's information such as rank per category, see your own stats, and view the complexity and category of a map.
## Required Mods
- BSIPA
- Leaderboard Core
- Beat Saber Markup Language (BSML)
- BS Utils
## Installation
Simply go to the releases tab, find the latest version, and then pick that zip file that matches your version. Then, take the dll inside and put it into your "Plugins" folder.
## Supported game versions
- 1.29.1
- 1.39.0
- 1.40.8
